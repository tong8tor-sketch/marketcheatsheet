# Market Cheat Sheet 📈

Una página web sofisticada y moderna para enumerar instrumentos de inversión en el mercado argentino.

## 🎯 Objetivo

Proporcionar una guía visual e interactiva de los principales instrumentos de inversión disponibles, clasificados por tipo y nivel de riesgo.

## 📊 Instrumentos Incluidos

- **Dólar** - Moneda extranjera de referencia
- **Acciones** - Participación accionaria en empresas
- **Bonos** - Títulos de deuda estatal
- **CEDEARs** - Acciones internacionales en pesos
- **Letras** - Títulos de corto plazo
- **Obligaciones Negociables** - Deuda corporativa
- **Fondos Comunes de Inversión** - Carteras diversificadas
- **Cauciones** - Préstamos de valores garantizados

## 🎨 Características de Diseño

### Interfaz Sofisticada
- **Gradientes modernos**: Colores complementarios con transiciones suaves
- **Animaciones elegantes**: Efectos de entrada y hover interactivos
- **Diseño responsivo**: Se adapta perfectamente a cualquier dispositivo
- **Iconografía clara**: Cada instrumento tiene un icono representativo

### Funcionalidades
- ✅ Filtrado por categoría (Renta Fija, Renta Variable, Otros)
- ✅ Tarjetas interactivas con información detallada
- ✅ Navegación suave entre secciones
- ✅ Indicadores visuales de riesgo
- ✅ Sección informativa adicional
- ✅ Optimización para móvil

## 🚀 Tecnologías

## 🎯 Categorías de Inversión

### Renta Fija
Instrumentos con rentabilidad predecible:
- Bonos
- Letras
- Obligaciones Negociables
- Cauciones

### Renta Variable
Instrumentos con potencial de crecimiento:
- Acciones
- CEDEARs

### Otros
- Dólar (Moneda)
- Fondos Comunes de Inversión (Vehículos)

## 📱 Características Responsive

La página se adapta automáticamente a diferentes tamaños de pantalla:
- 📱 Móvil (< 768px)
- 💻 Tablet (768px - 1024px)
- 🖥️ Desktop (> 1024px)

## 💡 Cómo Usar

1. Abre `index.html` en tu navegador
2. Explora los diferentes instrumentos en la galería
3. Usa los filtros para categorizarlos por tipo
4. Haz clic en cada tarjeta para ver más detalles
5. Lee la sección informativa para consejos de inversión

## ⚠️ Disclaimer

Esta página es solo informativa y educativa. No constituye asesoramiento financiero. 
Consulta con expertos antes de realizar cualquier inversión.
